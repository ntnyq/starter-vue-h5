# starter-vue-h5

Vue 移动端项目模板，组件库 Vant.

## Project setup

```bash
$ yarn install
```

### Compiles and hot-reloads for development

```bash
$ yarn run serve
```

### Compiles and minifies for production

```bash
$ yarn run build
```

### Lints and fixes files

```bash
$ yarn run lint:fix
```
